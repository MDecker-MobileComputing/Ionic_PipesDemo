# Ionic-App "Pipe-Demo" #

<br>

Diese Repository enthält ein [Ionic](https://ionicframework.com/)/Angular-Projekt für eine mobile App,
welches die Verwendung von [Angular-Pipes](https://angular.io/guide/pipes) demonstriert.
